Para poder jugar a la Batalla Naval es necesario tener 2 computadoras conectadas en LAN y una serie de bibliotecas instaladas que se detallaran abajo.
Una de las computadoras hara de servidor y cliente y la otra solo de cliente. Primero se debe checkear la direccion ip de ambas computadoras,
para hacer esto simplemente se abre la consola de windows y se escribe "ipconfig". Para saber cual es la direccion ip que se debe utilizar
nos fijaremos que sea del formato 192.168.191.x. Una vez encontrada la computadora que hace de servidor debe copiar su direccion ip en el 'server adress' del
archivo servidor y del archivo client, la otra computadora solo debe hacerlo en el archivo client.
